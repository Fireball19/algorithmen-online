ShakerSort

ShakerSort hat eine Laufzeit von O(n) im Best-Case und von O(n²) im Average- und Worst-Case.

Demo


Pseudocode

ShakerSort (Element A[])
1 boolean vertauscht;
2 int m := -1;
3 int n := A.length() - 1;
4 do
5   vertauscht := false;
6   m := m + 1;
7   for (int i := m; i < n; i++) do
8    if (A[i].k > A[i + 1].k) then
9     vertausche (A, i, i + 1);
10     vertauscht := true;
11    end
12   end
13   if (vertauscht == false) then
14    break
15   end
16   vertauscht := false;
17   n := n - 1;
18   for (int i := n - 1; i > m - 1; i--) do
19    if (A[i].k > A[i + 1].k) then
20     vertausche (A, i, i + 1);
21     vertauscht := true;
22    end
23   end
24 while (vertauscht == true);
