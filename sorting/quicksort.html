QuickSort

QuickSort hat eine Laufzeit von O(n log(n)) im Best- und Average-Case und von O(n²) im Worst-Case.

Demo


Pseudocode

int l := 0;
int r := A.length() - 1;
QuickSort (Element A[], int l, int r)
1 if (l < r) then
2  int p := r;
3  int i := l;
4  int j := r - 1;
5   do
6    while ((i <= j) and (A[i].k <= A[p].k)) do
7     i++;
8    end
9    while ((i <= j) and (A[j].k >= A[p].k)) do
10     j--;
11    end
12    if (i < j) then
13     vertausche(A, i, j);
14     i++;
15     j--;
16   while (i <= j)
17   vertausche(A, i, r);
18   QuickSort(A, l, i - 1);
19   QuickSort(A, i + 1, r);
20  end
