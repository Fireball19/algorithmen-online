Hashingalgorithmus Hashfunktion Sondierungsfolge
Lineares Sondieren k mod n h(k), h(k) − 1, h(k) − 2, ..., 0, n − 1, ..., h(k) + 1
Quadratisches Sondieren k mod n h(k), (h(k) + 1) mod n, (h(k) − 1) mod n, (h(k) + 4) mod n, (h(k) − 4) mod n, ...
Double Hashing h_0(k) := k mod n,
h_1(k) := 1 + (k mod (n − 2)),
n ist Primzahl		 h_0(k), (h_0(k) − h_1(k)) mod n, (h_0(k) − 2 · h_1(k)) mod n, ..., (h_0(k) − (n − 1) · h_1(k)) mod n
Brents Algorithmus h_0(k) := k mod n,
h_1(k) := 1 + (k mod (n − 2)),
n ist Primzahl		 Versuche einfügen bei i := h_0(k), sonst
betrache b_0 := (i - h_1(k)) mod n, sonst
betrache b_1 := (i - h_1(k_1)) mod n, wobei k_1 := Schlüssel auf Position i, setze k auf Position i und k_1 auf Position b_1, sonst
wiederhole ab b_0 mit i := b.
